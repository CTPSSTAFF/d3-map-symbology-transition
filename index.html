D3 Map Symbology Transition: Electric/Gas/Cable Utility Providers


Data sources: GeoJSON and CSV

  1. Spatial data loaded from GeoJSON file.
    • Derived from a clipped version of MassGIS TOWNSURVEY_POLYM layer.
    • GeoJSON first generated in ArcMap using Calvin Metcalf's esri-to-open tool.
  2. Tabular data loaded from CSV file - MassGIS TOWNS_POLY_UTILITIES table.
  3. The 'join' of #1 with #2 is implemented (trivially) in JavaScript - see comments in the code.
  4. Includes simple D3 on-hover "title" tooltip for each town.
  5. Pan and zoom (from 0.2x to 5x) - Not yet implemented.
