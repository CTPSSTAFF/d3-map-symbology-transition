D3 Map Symbology Transition: Electric/Gas/Cable Utility Providers


Data sources: TopoJSON and CSV

  1. Spatial data loaded from TopoJSON file.
    • Derived from a clipped version of MassGIS TOWNSURVEY_POLYM layer.
    • GeoJSON first generated in ArcMap using Calvin Metcalf's esri-to-open tool.
    • GeoJSON then converted to TopoJSON using geojson.io web service.
    • TopoJSON manipulated using Mike Bostock's topojson.js library.
  2. Tabular data loaded from CSV file - MassGIS TOWNS_POLY_UTILITIES table.
  3. Loading of #1 and #2 managed by Mike Bostock's queue.js library.
  4. The 'join' of #1 with #2 is implemented (trivially) in JavaScript - see comments in the code.
  5. Includes simple D3 on-hover "title" tooltip for each town.
  6. Pan and zoom (from 0.2x to 5x)
Symbolize by utility provider: Electric Gas Cable
